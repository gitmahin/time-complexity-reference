## Node.js time complexity

Node.js runs on the V8 engine and inherits all JavaScript built-in methods. Understanding their time complexity helps you write efficient server-side code - specially when handling large datasets, streams, or high-throughput APIs.

> All complexities listed are average case. Hash-based structures (Map, Set, Object) assume minimal collisions.
