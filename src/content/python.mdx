## Python time complexity

A reference guide for the time complexity of Python built-in data structures — list, dict, set, deque, and string. Python's `dict` and `set` are hash-based and provide O(1) average case for most operations.

> Python's `list` is a dynamic array — `append()` is O(1) amortized but `insert(0, x)` and `pop(0)` are O(n). Use `collections.deque` for efficient front operations.
