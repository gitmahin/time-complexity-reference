## C++ time complexity 

A reference guide for the time complexity of C++ STL containers and string methods. The STL provides both ordered (tree-based) and unordered (hash-based) variants - choosing the right one can significantly impact performance.

> Ordered containers like `std::map` and `std::set` use red-black trees giving O(log n) operations. Unordered variants like `std::unordered_map` give O(1) average but O(n) worst case.
