## C# time complexity

A reference guide for the time complexity of C# collections — List, Dictionary, HashSet, and SortedDictionary. The .NET runtime provides both hash-based and tree-based variants optimized for different use cases.

> `Dictionary<K,V>` provides O(1) average lookups. Use `SortedDictionary<K,V>` when you need keys in sorted order at the cost of O(log n) operations.
