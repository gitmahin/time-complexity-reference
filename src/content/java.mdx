## Java time complexity

A reference guide for the time complexity of Java Collections Framework — including ArrayList, HashMap, TreeMap, HashSet, and LinkedList. Java provides both hash-based and tree-based implementations for most data structures.

> Prefer `HashMap` over `TreeMap` when order doesn't matter — O(1) vs O(log n). Use `LinkedList` only when frequent insertions at the head are needed.
