## Rust time complexity

A reference guide for the time complexity of Rust standard library collections — Vec, HashMap, HashSet, and BTreeMap. Rust's ownership model guarantees memory safety without a garbage collector, making these complexities predictable.

> `HashMap` uses a high-quality hash function by default. Use `BTreeMap` when you need sorted keys or range queries — it provides O(log n) operations backed by a B-tree.
