## C time complexity

A reference guide for the time complexity of C standard library functions. C has no built-in data structures — complexity depends on how you implement them using arrays, pointers, and manual memory management.

> C arrays provide O(1) random access but no dynamic resizing. Use `qsort` for sorting and `bsearch` for binary search on sorted arrays.
